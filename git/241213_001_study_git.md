## Vim Command

- normal mode
  - i: insert mode 로 진입
  - ESC: back to normal mode

- insert mode
  - :q - quit
  - :q! - quit not write(save)
  - :w - write (save)
  - :wq - write and quit
  - :{num} - go to {num}th line
