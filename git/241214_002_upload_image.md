# md 파일에 이미지 추가 (use Issues)

- Github의 Issues 탭으로 이동
- Add a description - Write 공간에 이미지를 드래그 하거나 복사해 둔 이미지를 붙여넣기 한다
- 이미지가 업로드 된 후 생성된 마크다운 이미지경로를 md파일에서 사용
